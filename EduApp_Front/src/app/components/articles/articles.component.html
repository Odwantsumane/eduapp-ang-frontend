    Git

    Git is a distributed version control system that tracks versions of files. It is often used to control source code by programmers who are developing software collaboratively There are three commands with similar names: git reset, git restore and git revert. git-revert is about making a new commit that reverts the changes made by other commits. git-restore is about restoring files in the working tree from either the index or another commit.
